# 你的手艺人格是什么？

中国乡村工匠焕青计划 H5 测试游戏。

这个 H5 用 10 道生活化选择题，帮年轻人找到更适合走近的乡村手艺方向，并生成一张可分享的国潮大师人格卡。它不是身份认定，也不承诺速成，而是一个通向真实工匠老师、真实材料和真实学习的入口。

## 核心体验

- 10 道选择题，约 2 分钟完成
- 16 种手艺人格，对应 16 类乡村工匠手艺方向
- 结果页包含 MBTI、人格解释、推荐手艺、第一次学习建议和尊重提示
- 支持保存人格卡和系统分享
- 内置 16 位大师形象图鉴

## 本地运行

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
```

## 发布

推送到 `main` 后，GitHub Actions 会自动构建并部署到 GitHub Pages。

## 传播主张

> MBTI 告诉你是谁，这个测试告诉你：你该把自己交还给哪一门手艺。

更稳妥的活动表达是：

> 测测你适合从哪一门乡村手艺开始，跟真实乡村工匠老师学一件真正能留下来的东西。

## 真实性边界

- 不把所有乡村工匠称为非遗传承人
- 不写“测出你的非遗身份”
- 不承诺“一次学会”或“拜师成大师”
- 不把乡村作为城市情绪消费的背景板
- 不把民族纹样、服饰、银饰等做无出处拼贴
- AIGC 和视觉包装只做入口，真实工匠和真实手艺是主角

## 上线后建议关注

- 完成率
- 结果卡保存率
- 分享率
- 点击“想去学这门手艺”或后续报名入口的比例
- 结果分布是否过度集中
